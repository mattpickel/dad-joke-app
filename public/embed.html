Embedding the Dad Jokes app

Paste this snippet anywhere on your site. The iframe auto-resizes to fit content.

<iframe
  id="dadjokes-frame"
  src="https://YOUR-DEPLOYED-URL/"
  style="width:100%;border:0;display:block;"
  title="Dad Jokes"
  loading="lazy"
></iframe>
<script>
  window.addEventListener('message', function (e) {
    if (e.data && e.data.type === 'dadjokes:height') {
      var f = document.getElementById('dadjokes-frame');
      if (f) f.style.height = e.data.height + 'px';
    }
  });
</script>

Notes

  • Replace YOUR-DEPLOYED-URL with the live deployment (e.g. https://dadjokes.sumner.kids).
  • The iframe sends its content height to the parent page via postMessage. The script above resizes it to match — so there's no scrollbar inside the iframe.
  • If you want a fixed height instead of auto-resize, drop the <script> block and set height: 800px (or whatever you prefer) on the iframe style.
  • For stricter security, replace the '*' origin check in the snippet with your deployed origin.
